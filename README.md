# 🏫 AI-Smart Campus System
### 👥 Official Team Name: CSE4204-8A-T07 (Section: 8A)

---

## 📝 Project Description
The **AI-Smart Campus System** is an intelligent, integrated academic management and student success platform designed for Northern University of Business and Technology, Khulna. Moving away from traditional, passive Learning Management Systems (LMS) that act merely as data repositories, this platform introduces real-time academic decision support, automated progress monitoring, and interactive workflows. By joining a robust frontend with an enterprise backend and localized AI pipelines, the platform enhances institutional productivity and improves student learning efficiency.

---

## 🎯 Problem Statement
Modern university students frequently struggle to manage fragmented academic information, track critical performance indicators like CGPA or attendance, and stay updated with rapid notice updates. At the same time, faculty members find it difficult to efficiently monitor student engagement vectors and identify academically vulnerable students early enough for timely intervention. 

Current institutional portals fail to deliver personalized recommendations or automated insights. While public AI tools exist, they lack safe connectivity to private university routines, course lines, and continuous evaluation marks. To resolve this technological gap, our project integrates local relational schemas with modern frameworks to introduce localized chatbot guidelines, predictive risk identifiers, and proactive course advice.

---

## 🎯 Project Objectives
* **Centralized Operations:** To build a smart, all-in-one student productivity portal tightly connected to campus workflows.
* **Personalized Academic Guidance:** To generate optimized study paths and steering strategies based on historical grading and performance trends.
* **Instant Conversational Support:** To integrate a 24/7 AI Academic Assistant chatbot to address information access issues and answer FAQs.
* **Proactive Risk Intervention:** To maximize student success and minimize dropout rates through automated predictive grade warning flags.

---

## 🚀 Proposed Features

### 🧠 Core AI Features & Workflows
* **AI-Powered Academic Course Advisor:** Continuously evaluates a student's prior academic performance records, strengths, and specific semester histories within the MySQL layer to deliver data-driven course suggestions for future semesters.
* **Student At-Risk Detection & Performance Analytics:** Evaluates ongoing engagement parameters including attendance rates, quiz marks, task deliverables, and mid-term scores to dynamically trigger automated high-risk warnings for vulnerable students.
* **AI Campus Assistant (Gemini API):** An NLP-driven, persistent chat interface utilizing system prompt profiling to provide immediate, context-bounded answers regarding official exam schedules, notice boards, library hours, and campus FAQs.

### 💻 General System Modules
* **Secure Authentication:** Multi-role gateway infrastructure enforcing role-based user authentication using JWT tokens tailored for students, faculty, and administrative staff.
* **Smart Performance Dashboard:** An interactive hub displaying clean visual progress charts to track student attendance logs, quiz tracking distributions, and mid-term grades.
* **Administrative Control Panel:** A master interface facilitating profile lifecycle checks, account approval management, platform settings, and global information broadcasts.

## 🛠️ Technology Stack
* **Frontend Presentation Layer:** React.js, Tailwind CSS (Formulated for fully responsive mobile and web layouts)
* **Backend Framework:** Laravel Framework
* **Database Infrastructure:** MySQL (XAMPP Environment)
* **AI Services Layer:** Gemini API / OpenAI API / HuggingFace (Utilizing custom system profiling and prompt engineering)
