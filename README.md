# 🏫 AI-Smart Campus System
### 👥 Official Team Name: CSE4204-8A-T07 (Section: 8A)

---

## 📝 Project Description
[cite_start]The **AI-Smart Campus System** is an intelligent, integrated academic management and student success platform designed for Northern University of Business and Technology, Khulna[cite: 101, 112]. [cite_start]Moving away from traditional, passive Learning Management Systems (LMS) that act merely as data repositories, this platform introduces real-time academic decision support, automated progress monitoring, and interactive workflows[cite: 109, 126, 198]. [cite_start]By joining a robust frontend with an enterprise backend and localized AI pipelines, the platform enhances institutional productivity and improves student learning efficiency[cite: 114, 119].

---

## 🎯 Problem Statement
[cite_start]Modern university students frequently struggle to manage fragmented academic information, track critical performance indicators like CGPA or attendance, and stay updated with rapid notice updates[cite: 107]. [cite_start]At the same time, faculty members find it difficult to efficiently monitor student engagement vectors and identify academically vulnerable students early enough for timely intervention[cite: 108]. 

[cite_start]Current institutional portals fail to deliver personalized recommendations or automated insights[cite: 110]. [cite_start]While public AI tools exist, they lack safe connectivity to private university routines, course lines, and continuous evaluation marks[cite: 127]. [cite_start]To resolve this technological gap, our project integrates local relational schemas with modern frameworks to introduce localized chatbot guidelines, predictive risk identifiers, and proactive course advice[cite: 128, 131].

## 🎯 Project Objectives
* [cite_start]**Centralized Operations:** To build a smart, all-in-one student productivity portal tightly connected to campus workflows[cite: 169].
* [cite_start]**Personalized Academic Guidance:** To generate optimized study paths and steering strategies based on historical grading and performance trends[cite: 170].
* [cite_start]**Instant Conversational Support:** To integrate a 24/7 AI Academic Assistant chatbot to address information access issues and answer FAQs[cite: 171, 206].
* [cite_start]**Proactive Risk Intervention:** To maximize student success and minimize dropout rates through automated predictive grade warning flags[cite: 172].

## 🚀 Proposed Features

### 🧠 Core AI Features & Workflows
* [cite_start]**AI-Powered Academic Course Advisor:** Continuously evaluates a student's prior academic performance records, strengths, and specific semester histories within the MySQL layer to deliver data-driven course suggestions for future semesters[cite: 136, 141].
* [cite_start]**Student At-Risk Detection & Performance Analytics:** Evaluates ongoing engagement parameters including attendance rates, quiz marks, task deliverables, and mid-term scores to dynamically trigger automated high-risk warnings for vulnerable students[cite: 143, 148].
* [cite_start]**AI Campus Assistant (Gemini API):** An NLP-driven, persistent chat interface utilizing system prompt profiling to provide immediate, context-bounded answers regarding official exam schedules, notice boards, library hours, and campus FAQs[cite: 154, 179].

### 💻 General System Modules
* [cite_start]**Secure Authentication:** Multi-role gateway infrastructure enforcing role-based user authentication using JWT tokens tailored for students, faculty, and administrative staff[cite: 174].
* [cite_start]**Smart Performance Dashboard:** An interactive hub displaying clean visual progress charts to track student attendance logs, quiz tracking distributions, and mid-term grades[cite: 174].
* [cite_start]**Administrative Control Panel:** A master interface facilitating profile lifecycle checks, account approval management, platform settings, and global information broadcasts[cite: 174].
